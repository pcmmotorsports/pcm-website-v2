## 3. 16c 要做什麼(實作 scope)

### 3.1 🔴 商品卡真圖(修 Sean 肉眼驗到的「通用佔位照」根因)
- **`toUIProduct`(`apps/storefront/src/lib/products.ts` L78-108)目前無 image 欄** —— 只設 `imgTone:'neutral'` 佔位,`ProductImage` 用 seed 生成通用照(這就是首頁 featured 顯機車生活照、不顯真碳纖維照的根因)。
- 修法:UI 型別(`MockProduct`)+ `toUIProduct` **加 image 欄 ← `product.images[0]`**(群代表圖);`ProductCard` / `ProductImage` 改渲染真圖、無圖才 placeholder。
- 影響 featured / ProductsPage / BrandIndex / VehicleFinder 所有商品卡。

### 3.2 詳情頁接真商品 + by-handle 查詢
- **`findByHandle` 缺口**:`SupabaseProductAdapter` 只有 `findById`(5 read method),詳情頁路由是 `apps/storefront/src/app/products/[slug]/page.tsx`(slug = handle)→ **加 `findByHandle(handle)`**(仿 findById、`.eq('handle', h)`)。
- `lib/products.ts` 加 `fetchProductByHandle(handle, tier)`。
- 詳情頁 `[slug]/page.tsx` 由 mock 換真(`fetchProductByHandle`)。

### 3.3 adapter 接變體(backlog #203、本片落地)
- `PRODUCT_SELECT_DETAIL` embed **`product_variants_public(...)`**(🔴 走 view、不含 price_store/metadata)。
- 加 `mapVariantRow`:DB `price_general`/`price_store` 兩整數欄 → domain `ProductVariant.priceByTier`(general 從 price_general、**store 走 dummy、premiumStore placeholder** —— 與 `mapSupabaseProductToDomain` 同法、見現有 `mappers/product.ts` L129-151)。
- `findByHandle` / save 帶變體;list 路徑**不帶變體**(避 N+1 jsonb 膨脹)。
- **contract test**:測 `product_variants_public` DDL 投射不含 price_store/metadata + mapVariantRow 不洩經銷價(working-style 第 35 條)。

### 3.4 ProductInfo 去 hardcode、吃真變體
- `apps/storefront/src/components/ProductInfo.tsx`(`'use client'`)目前 hardcode:`COLOR_MAP`(8 色 L34)+ `sizeOptions`(4 category 分支 L47)+ `colorOptions`(L58)+ reset useEffect(L76-82)。**鐵則 9 L3 已標**。
- 改吃真 `product.variants`:**weave selector + finish selector**(從 variant.spec);選了 → `currentVariant`(比對 spec)→ 換 price / images / availability。
- `MockProduct` / `toUIProduct` **加 variants 欄**(不是換型別、是補 variants + 去 hardcode 補洞;rename UIProduct 屬獨立 cleanup、別混)。

### 3.5 🔴 經銷價防護(16c 最重要、審查盯死)
- **變體 price 也要 server-side strip 成單一值**(`computeEffectivePrice(product, variant, tier)`),`priceByTier` 整包**不進 client bundle**。
- 變體的 premiumStore 計算需**父商品 brand 的 `premium_extra_pct`**(變體不帶 brand)→ 取價時帶父 brand context。
- **commit 前 grep client bundle**:無 `price_store` / `shopee` / `cost` / `price_by_tier` / `metadata` 外洩。
- adapter 只 select `*_public` view、絕不 select base 表敏感欄。
