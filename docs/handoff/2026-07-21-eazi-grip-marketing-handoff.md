## 4. 關鍵事實與數字(可驗證,勿憑記憶改)

### 成本結構(x = 原廠 GBP 定價)

```
現行(38% off):cost = x * 0.63 * 45 * 1.15 = 32.6x
跨 £5001 後(45% off):cost = x * 0.55 * 45 * 1.15 = 28.5x
```

- 原廠折扣級距:**£5000 以下 38% off;£5001-15000 為 45% off,享年度追溯返利**(回溯適用,不需一次押大單)
- 跨過門檻約需 **177 組/年 ≈ 每月 15 組**,跨過後每組毛利永久增加約 190 元
- 英國**空運 2-3 週**到貨(非海運)
- 代理合約**無年度 MOQ**

### 🔴 系統成本公式是錯的,必須修

資料庫現行:`round(x * 0.557 * 50)` = 27.85x → **低估真實成本 17%**,所有毛利報表失真。

正確:`round(x * 0.63 * 45 * 1.15)`

### 定價公式(Codex 提出,分析後認為方向正確)

```
general = round(x * 52 / 50) * 50
store   = round((round(x * 52 / 50) * 50) * 0.8 / 50) * 50
shopee  = ceil((round(x * 52 / 50) * 50) * 1.06 / 10) * 10
cost    = round(x * 0.63 * 45 * 1.15)     ← 已修正
```

以中位數品項(x ≈ 45.6)驗算:零售 2,350 / 車行 1,900 / 蝦皮 2,500 / 成本 1,487。
毛利:零售 36.7%、車行 21.7%;跨 £5001 後為 44.8% / 31.7%。

**52x 的依據**:剛好等於 steadymoto 站上 Eazi-Grip 的中位數 2,350(見下)。

**蝦皮 1.06 未驗證** — 需查蝦皮後台實際費率(成交手續費 + 金流費合計)。若實際 7-8%,此係數要調高。

### 競品實況(已查證)

| 對象 | 售價 | 性質 |
|---|---|---|
| 蝦皮 rexsunnoriko | 統一 1,328,鋪滿數十車款 | 低於 PCM 成本 → 幾乎確定清倉 |
| **steadymoto.tw(台中馳穩,統編 60228000)** | **Eazi-Grip 44 筆,1,050-3,450,中位數 2,350** | **自稱官方授權代理,掛在 R&G 分類下** |
| steadymoto R&G 自有 Tank Traction Grips | 127 筆,950-3,450,中位數 1,800 | 不同品牌,不可直接比 |
| TechSpec | 2,400-3,450 | 有台灣代理經營 |
| Stompgrip | 2,750-2,900 | — |

**PCM 相對優勢**:料號 **5,243 筆 vs steadymoto 44 筆**;steadymoto 全站查無「施工/代客安裝」字樣;其多筆商品標示缺貨。

## 6. 風險與未解

| 狀態 | 項目 |
|---|---|
| ✅ 已解除 | 年度 MOQ 壓垮現金流(合約無 MOQ) |
| ✅ 大幅緩解 | 進貨撐不起 KPI(空運 2-3 週可滾動補貨,非海運 4-8 週) |
| 🔴 **新增、最嚴重** | steadymoto 可能是**結構性長期競爭者**而非清倉,取決於 R&G 供貨權(待辦 #1) |
| 🟡 需持續驗證 | 「水貨會賣完」。**驗證法**:每週固定日截圖蝦皮前 5 大水貨賣家的已售出數與在架車款數,連續 4 週。若售出遞增但在架車款不減 → 有人在補貨,前提證偽,廣告策略改道 |
| 🟡 未解決 | 車行毛利 21.7% 偏薄。**話術對策**:對車行講「一單實拿 950」(450 價差 + 500 工錢),不要講「8 折」 |
| 🟡 時間點 | 7-8 月是台灣重機淡季。建議定為準備期(做板、簽車行、拍素材、驗證),廣告火力延到 9 月 |
| ⚪ 提醒 | 台灣對真品平行輸入原則上不構成商標侵權,**「總代理授權」不能用來讓蝦皮下架水貨賣家**。授權聲明只用於建立自己的信任,不作為打擊工具 |
