### 2.4 ProductInfo.tsx — 去 hardcode、吃真變體(🔴 共用元件)
- **移除**:COLOR_MAP、sizeOptions useMemo、colorOptions useMemo、color/size useState、reset useEffect 的 color/size 部分。
- **加**:
  - `SPEC_LABELS`:{ weave: {Forged:'鍛造紋',...}, finish: {Glossy:'亮面',Matt:'消光'}, special: {'12K':'12K碳纖',Kevlar:'Kevlar芳綸'} } + key 顯示名 { weave:'紋路', finish:'表面', special:'特殊材質' }。無對照則顯原值(fallback)。
  - `specKeys` 派生:variants 的 spec key 聯集、固定序 weave→finish→special→其他(只渲染 distinct 值 >1 的 key;Q3=A 資料驅動)。
  - `selectedVariant` useState(預設 variants[0];product 變時 reset、useEffect [product.slug])。**LOCAL state**(16c-4 才提升 ProductPage)。
  - selector 渲染:每個 specKey 一個 `.pd-opt`(label = key 中文 + 當前值中文)、`.pd-size-grid` 內每個 distinct 值一個 `.pd-size-btn`(顯中文、aria-pressed = 是否當前選中);Q4=A 不顯庫存、不 disable。
  - 選擇 handler `selectSpec(key, value)`:候選 = variants.filter(spec[key]===value);選「與當前 selectedVariant 其他維度相符最多」者(tie-break sortOrder/原序);snap 保證永遠有效變體(稀疏矩陣不卡死)。
  - 價顯示:`displayPrice = selectedVariant?.price ?? product.price`;`.pd-price` 顯 displayPrice。tier 經銷 tag 區塊**維持原樣**(用 product.origPrice/tier、general 不顯 tag;變體無真經銷價、tier-aware 變體價延 M-2-08、不在本片)。

- **addToCart(cart 處置、不改 CartContext 契約)**:原送 {productId:slug, qty, color, size}。改送 `color = 變體規格中文摘要`(例「鍛造紋 · 亮面」或含 special)、`size = null`。
  - 理由:CartLineKey={productId,color,size} 本就是 variant 誤殺防線;變體規格摘要當 discriminator → 不同變體 = 不同 cart line + cart 顯人類可讀。**不改 CartContext 契約**(避 ripple、scope 控)。proper variantSku cart key 留 backlog follow-up。
